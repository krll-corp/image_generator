MNIST Digit Generator API

This is the API server for the MNIST Digit Generator.

To use the interactive frontend, please visit http://localhost:3000

Available API Endpoints:

  • /api/generate_conv_digit?digit=5 - Generate using ConvGenerator
  • /api/generate_vq_digit?digit=3 - Generate using VQ-Transformer
  • /api/generate_vq_vae_digit?digit=7 - Generate using VQ-VAE directly
  • /api/stream_digit?digit=2 - Stream generation (pixel or token-based)
